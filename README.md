# Senior Game Rental System

This project is a MySQL database system for a senior game rental platform.

## Files
- `GameBridge.sql` - database schema
- `sample_data.sql` - sample data
- `queries.sql` - analysis queries
- `Senior_Game_Rental_Report.pdf` - final report

## How to run
1. Install MySQL Server and MySQL Workbench
2. Open MySQL Workbench
3. Run `GameBridge.sql`
4. Run `sample_data.sql`
5. Run `queries.sql`
